# ⚡ Smart Home Energy Predictor — UI

A Streamlit web interface for interacting with the Smart Home Energy 
Prediction API. Simulates a Nigerian home's smart meter control panel 
where users can input readings and get power source recommendations.

## Built With
- Streamlit
- Plotly
- Python 3

## Features
- Input power meter readings and source status
- Predict next hour energy consumption
- Get recommended power source with colour coded result
- View consumption trend chart across multiple predictions
- Track full prediction history table

## How To Use
1. Open the app
2. Fill in meter readings and power source status
3. Click ⚡ Predict Next Hour Consumption
4. View prediction and recommended power source

## Power Source Colour Coding
| Colour | Source |
|---|---|
| 🟢 Green | Grid |
| 🔵 Blue | Solar |
| 🟡 Yellow | Generator |
| 🔴 Red | Load Shedding |
